# 재능기부 매칭 플랫폼 프로젝트 계획

## 프로젝트 개요
사람들이 자신의 재능을 나누고 다른 사람의 재능을 배울 수 있는 재능기부 매칭 플랫폼

## 핵심 가치
- 무료 재능 공유를 통한 사회 기여
- 서로 배우고 가르치는 커뮤니티 형성
- 접근성 높은 교육 기회 제공

## MVP (최소 기능 제품) 기능 목록

### 1. 재능 제공자 기능
- **재능 등록**
  - 제공할 재능/기술 제목
  - 간단한 설명
  - 가능한 일정 등록 (날짜 + 시간)
  - 장소 또는 온라인 여부
  - 최대 수강 인원

### 2. 재능 수혜자 기능
- **수업 목록 조회**
  - 전체 재능 목록 보기
  - 카테고리별 필터 (예: 언어, 프로그래밍, 요리, 운동 등)
  - 일정별 필터
- **수업 신청**
  - 원하는 수업 선택
  - 가능한 일정 중 선택하여 신청

### 3. 기본 기능
- **간단한 사용자 인증**
  - 이름, 이메일 정도의 기본 정보
  - 로그인/로그아웃
- **내 활동 관리**
  - 내가 등록한 재능 목록
  - 내가 신청한 수업 목록
  - 내 재능에 신청한 사람 목록

## 기술 스택 (제안)

### Frontend
- **React** - 사용자 인터페이스
- **React Router** - 페이지 라우팅
- **Axios** - API 통신
- **CSS Framework** (선택): Tailwind CSS 또는 Material-UI

### Backend
- **Node.js + Express** - API 서버
- **MongoDB** 또는 **PostgreSQL** - 데이터베이스
- **JWT** - 간단한 인증

### 배포 (선택)
- Frontend: Vercel 또는 Netlify
- Backend: Render 또는 Railway
- Database: MongoDB Atlas 또는 Supabase

## 데이터 모델

### User (사용자)
```
- id
- name
- email
- password (해시)
- createdAt
```

### Talent (재능/수업)
```
- id
- userId (등록한 사용자)
- title (제목)
- description (설명)
- category (카테고리)
- location (장소 또는 "온라인")
- maxParticipants (최대 인원)
- schedules (일정 배열)
- createdAt
```

### Schedule (일정)
```
- id
- talentId
- date (날짜)
- startTime (시작 시간)
- endTime (종료 시간)
- currentParticipants (현재 신청 인원)
```

### Booking (예약/신청)
```
- id
- userId (신청한 사용자)
- talentId
- scheduleId
- status (대기/확정/취소)
- createdAt
```

## 주요 페이지 구성

1. **홈페이지**
   - 프로젝트 소개
   - 최근 등록된 재능 미리보기
   - 시작하기 버튼

2. **재능 목록 페이지**
   - 모든 재능 카드 형식으로 표시
   - 필터 (카테고리, 날짜, 온라인/오프라인)
   - 검색 기능

3. **재능 상세 페이지**
   - 재능 정보 상세
   - 가능한 일정 목록
   - 신청하기 버튼

4. **재능 등록 페이지**
   - 재능 정보 입력 폼
   - 일정 추가 기능

5. **마이페이지**
   - 내가 등록한 재능
   - 내가 신청한 수업
   - 내 재능에 신청한 사람들

6. **로그인/회원가입 페이지**
   - 간단한 인증 폼
